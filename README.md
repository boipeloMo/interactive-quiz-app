# Interactive Quiz App

A responsive JavaScript mini app that pairs a simple quiz experience with a companion calculator, redesigned to feel cleaner, more modern, and much stronger as a GitHub portfolio project.

## Overview
This project started as a very minimal quiz page with a separate calculator page. It has been refactored into a mobile-first interface with better structure, spacing, typography, and responsive behavior across mobile, tablet, and desktop. The result is a much more polished beginner-friendly frontend project that still keeps the code approachable.

## Setup Instructions
This is a static frontend project, so no package installation is required.

### Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/boipeloMo/interactive-quiz-app.git
   ```
2. Open the project folder.
3. Open `index.html` in your browser.

### Run With VS Code Live Server
1. Open the folder in VS Code.
2. Right-click `index.html`.
3. Choose `Open with Live Server`.
